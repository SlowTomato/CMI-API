# CMI-API
Public API for CMI plugin


## Maven

```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>

<dependency>
    <groupId>com.github.SlowTomato</groupId>
    <artifactId>CMI-API</artifactId>
    <version>VERSION</version>
</dependency>
```
